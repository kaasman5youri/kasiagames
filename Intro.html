    Intro

    Waarom een unity game?

    We hebben gekozen voor een unity game omdat we wouden leren hoe we games in unity konden maken door C# te gebruiken.

    Wat voor game en hoe kregen we het idee voor de game?

    We hebben gekozen om een rpg te maken en we hebben gekozen voor de naam Soulbreaker. De dingen die hierin staan zijn ideeÃ«n en we hebben de meeste nog niet af gekregen. Het verhaal van de game speelt zich af op een eiland. In het verhaal heeft iemand de zielen van iedereen op het eiland gestolen en ze in kristalen gedaan om er iets mee te doen (we hebben nog niet verzonnen waarom hij het doet). Het doel van de game is om hem te verslaan en alle zielkristallen te slopen. Maar je komt ook monsters tegen die je moet verslaan. Als je een monster verslaat krijg je geld wat je kan gebruiken om wapens te kopen. Ook hadden we het idee voor meerdere beroepen waarbij je verschillende soorten aanvallen hebt zoals magic, ranged en melee. En we hadden ook het idee voor verschillende gebieden zoals een woestijn, een jungle, een grot en een vulkaan.

    Wat hebben we geleerd?

    We hebben geleerd hoe we C# kunnen gebruiken om een game te maken in Unity. We hebben verschillende websites gebruikt om het te leren. We hebben een C# tutorial gedaan in w3schools en we hebben een paar youtube videos gekeken. Tijdens het programeren hebben we stackoverflow en unity documentation gebruikt.

    Welke obstakels zijn we tegen gekomen en waarom hebben we het niet af gekregen?

    We hebben het niet optijd af gekregen omdat het project dat we hebben gekozen te groot is. Op school kunnen wij ook niet programeren wat zorgde voor vertraging. Tijdens het programeren zijn we verschillende obstakels tegengekomen. Voor Youri was de code voor de pathfinding van de vijanden een obstakel. Hij vond het moeilijk te begrijpen en wist niet hoe hij de code moest maken. Hij heeft een tijd geprobeerd zelf de code te schrijven maar het lukte niet, later hebben we de code van iemand anders gebruikt.
